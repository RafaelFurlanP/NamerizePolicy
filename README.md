## 4. Almacenamiento y Seguridad de Datos / Data Storage and Security / Armazenamento e Segurança de Dados
#### Español
**Almacenamiento Local:** Todos tus datos de contacto se almacenan localmente en tu dispositivo en una base de datos SQLite. No subimos, sincronizamos ni hacemos copias de seguridad de tus datos personales en servidores en la nube.

**Fotos:** Las fotos de contactos se comprimen y almacenan en el directorio privado de la app en tu dispositivo.

**Seguridad:** Tus datos están protegidos por las medidas de seguridad de tu dispositivo. Recomendamos usar funciones de seguridad a nivel de dispositivo como códigos de acceso o autenticación biométrica.

#### English
**Local Storage:** All your contact data is stored locally on your device in a SQLite database. We do not upload, sync, or backup your personal data to any cloud servers.

**Photos:** Contact photos are compressed and stored in your device's private app directory.

**Security:** Your data is protected by your device's security measures. We recommend using device-level security features like passcodes or biometric authentication.

#### Português
**Armazenamento Local:** Todos os seus dados de contato são armazenados localmente no seu dispositivo em um banco de dados SQLite. Não fazemos upload, sincronizamos nem fazemos backup de seus dados pessoais em servidores na nuvem.

**Fotos:** Fotos de contatos são compactadas e armazenadas no diretório privado do app no seu dispositivo.

**Segurança:** Seus dados são protegidos pelas medidas de segurança do seu dispositivo. Recomendamos usar recursos de segurança a nível de dispositivo, como códigos de acesso ou autenticação biométrica.
